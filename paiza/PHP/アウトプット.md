# 2022/12/25

### PHPとは
- 動的なwebサービスを作るのに適した言語
- webアプリケーション開発に使われる
- ファイル拡張子は、.php

### 初歩
- 開始タグと終了タグ
```php
<?php
// ここにコードを記述
?>
```
- 文字列の出力
  - echoを使う
```php
<?php
// 文の最後はセミコロンが必要;
echo "hello wrold";
?>
```

### PHPと HTMLの共存
- HTMLに埋め込む出来る
```php
// htmlタグの中に埋め込むことに気をつける
<b><?php
echo "hello world";
?></b>

<p>hogehoge<p>
```
