## 配列

### 配列の使い所
・並び順管理が必要なデータ処理
・エクセルのような複数業のデータ処理
・フォーム等の選択肢の管理（性別、都道府県、etc）
・座席の空席管理
・CSVデータの処理（二次元配列、連想配列）
・将棋のような盤面の表現（二次元配列）

### 配列への代入の仕方
```
$配列名 = array(要素１, 要素２, 要素３…);
```
- 値の取り出しには、print_r() 関数を使用する
```php
<?php
$array = array("戦士","侍","僧侶","魔法使い");
// この下で$arrayを出力してみよう
print_r($array);
?>

// 出力結果
Array
(
    [0] => 戦士
    [1] => 侍
    [2] => 僧侶
    [3] => 魔法使い
)
```
